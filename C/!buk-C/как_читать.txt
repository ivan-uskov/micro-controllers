
1. Для просмотра файлов HTML, DOC, CHM, HLP 
   используйте средства операционной системы и офиса.

2. Для просмотра файлов PDF используйте AcrobatReader (если он установлен),
   либо воспользуйтесь программой FoxitReader.exe (или view-pdf.exe )
   (неинсталлируемый просмотровщик pdf-файлов)

3. Для просмотра файлов DJVU используйте view-djvu.exe 
   (неинсталлируемый просмотровщик djvu-файлов)
